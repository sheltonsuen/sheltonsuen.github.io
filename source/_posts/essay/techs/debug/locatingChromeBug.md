# Locating Chrome Bug
When you are confident the problem you are facing is a bug related to Chrome, then you can use the steps below to locate more infomation about it.
* find the bug in [chrome bug management platiform](https://bugs.chromium.org/p/chromium/issues/detail?id=784059)
* then confirm it
* if it's fixed then you can find which version fix it using [When will ship](https://www.chromium.org/blink/when-will-a-fix-ship-in-chrome-stable-or-canary)
